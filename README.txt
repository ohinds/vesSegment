This program uses eig3Voulme MEX file. You need to complie to get MEX file that works in your computer/Operating system. To get your MEX file run below command in MATLAB.

mex eig3volume.c

This command will not work if you don't have c/c++ compiler installed. Visit http://www.mathworks.com/support/compilers/R2017a/ to get c/c++ compiler that matlab supports.
